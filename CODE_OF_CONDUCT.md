# 行为准则（Code of Conduct）

本项目采用 Contributor Covenant Code of Conduct（v2.1）。

## 我们的承诺

为了营造开放、友好的环境，我们承诺让所有参与者都能获得无骚扰的体验，不论其年龄、体型、可见或不可见的残障、族裔、性别特征、性别认同与表达、经验水平、教育程度、社会经济地位、国籍、个人外貌、种族、宗教或性取向。

## 我们的标准

有助于营造积极环境的行为包括（但不限于）：
- 尊重不同观点与经验
- 善意沟通与建设性反馈
- 专注于对社区最有利的事情
- 对其他社区成员保持同理心

不可接受的行为包括（但不限于）：
- 使用性暗示或歧视性言论、侮辱/人身攻击
- 公开或私下骚扰
- 未经许可发布他人隐私信息
- 其他不专业或不适当的行为

## 执行

如遇到违反行为准则的情况，请在 GitHub Issue/PR 中说明上下文，维护者会尽力跟进处理。

> 说明：本文件基于 Contributor Covenant v2.1 的精神做了简化版本。
